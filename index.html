Limpeza de dados em SQL

Este projeto de manipulação de dados envolve o tratamento de informações relacionadas a demissões em massa de empresas. Começamos com a criação de um banco de dados e uma tabela para armazenar os dados. Em seguida, duplicamos a tabela original e realizamos a remoção de duplicatas, identificando e tratando valores nulos e inconsistentes nas colunas, como "industry", "country" e "date". Foi realizada uma padronização das informações para garantir que os dados fossem consistentes, como a correção de valores em branco e a atualização de campos com dados incompletos. Também foram feitas verificações e correções em registros nulos ou inconsistentes, com a exclusão de linhas que não contribuiam para a análise. No final, limpamos a tabela, removendo colunas desnecessárias e garantindo que os dados estivessem prontos para análises futuras. Este processo assegura a qualidade e a integridade dos dados, permitindo uma análise mais precisa sobre o impacto das demissões nas empresas.

dados em SQL

Este projeto foi voltado para a análise e limpeza de dados sobre casos de dengue. Utilizando um banco de dados MySQL, foram realizadas verificações de duplicidade, valores nulos e inconsistências nos dados, como a padronização de datas e eliminação de espaços extras. Além disso, foram gerados KPIs para analisar a distribuição dos casos de dengue por sexo, faixa etária, região e estado. Também foi realizada uma análise de tendência ao longo do tempo e proporções entre os casos de homens e mulheres. O objetivo foi garantir dados limpos e consistentes para uma análise mais precisa sobre a doença.

Visualização de
Dados em Power BI

Este dashboard de análise de conversão de leads apresenta uma visão abrangente do desempenho das campanhas de captação. Ele exibe métricas principais, como a quantidade de interessados, cliques e inscritos, juntamente com taxas de conversão detalhadas, incluindo taxa de cliques por disparo (2,0%), taxa de inscritos por disparo (25,7%) e taxa de conversão dos cliques (52,7%). Gráficos comparativos mostram o desempenho em termos de inscritos e interessados por diferentes testes de campanhas (como TPW teste A/B e VIP teste A/B), e a taxa de conversão é detalhada por copy, permitindo insights práticos para aprimorar estratégias de engajamento e conversão. A disposição das métricas e gráficos facilita uma análise clara e rápida, apoiando decisões mais assertivas para maximizar os resultados das ações de marketing.

EDA em Python
Séries de TV

Este projeto de Análise Exploratória de Dados (EDA) mergulha no mundo das séries de TV, utilizando dados do Kaggle para revelar padrões fascinantes. Através de gráficos e insights, investigamos a média das avaliações das séries, identificamos o gênero predominante, analisamos o ano com o maior número de lançamentos e destacamos os atores mais recorrentes. O estudo revela que as séries têm uma média de 7 pontos em avaliações, sendo o Drama o gênero mais comum, e 2022 foi o ano com mais lançamentos. Este projeto oferece uma visão abrangente sobre as tendências das séries de TV.
